What is this mod?

This is a mod intended to make Minecraft much harder then the base game.

Mechanics:

This is your thirst.

The farther it goes down, the more negative effects you receive until you start dying.

Easiest way to refill it is to drink water or potions.

Stuff:

Some text on the main menu has changed

No reused splash texts

Gamerules: easyMode and hardMode!

When creating a world, you can no longer select the gamemode, or difficulty.

Mobs around have been buffed, and some now have new abilites.

The stray's arrows are now ice arrows, which cause you to be frozen on hit, and create an ice area effect cloud if it misses.

Blaze's fireballs now create a fire area effect cloud on hit.

Zombies now come in groups of 1-5, all having equipment.

Some other stuff, but you get the point.
